# LunaTide (ルナタイド) 🌙✨

**LunaTide** は、天体観測、夜景撮影、釣り、そして月を愛するすべての人のために設計された、超高速・高精度の月齢＆潮汐情報プラットフォームです。

技術的な複雑さをバックエンドに閉じ込め、ユーザーには「速さ」と「心地よさ」という圧倒的な体験を提供することに特化しています。

---

## 🚀 圧倒的なパフォーマンス・特長

### 1. 瞬時のデータ検索 (Ultra-Fast Retrieval)
かつての重いCSV解析から、最適化された **SQLite データベース構造** へ移行。日本全国47都道府県、2025年〜2026年の膨大な天体データを、クリックした瞬間に表示します。待ち時間はほぼゼロ。あなたのインスピレーションを邪魔しません。

### 2. 滑らかな操作性 (Seamless UI/UX)
月齢シミュレーターは、高解像度の月画像を **バックグラウンドでインテリジェントにプリロード**。スライダーを動かすたびに、月が魔法のように滑らかに満ち欠けします。ページリロードなしで、直感的に未来や過去の月の姿を確認できます。

### 3. 日本全国・ピンポイント精度 (High Precision)
気象庁などの信頼できるソースを元にした精密な計算結果を保持。札幌から那覇まで、全国各地の正確な「月の出・月の入り・月齢」を提供します。秒単位のタイミングが重要な天体撮影やアクティビティを強力にサポート。

### 4. 視認性を極めたインテリジェント・カレンダー
重要な天体イベント（流星群、月食、惑星接近など）を自動でハイライト。「見逃せないイベント」がひと目で分かり、さらに独自の **「観測しやすさスコア」** や観測のコツも瞬時に提供します。

### 5. 軽量かつレスポンスの速いモダンデザイン
ダークモードを基調とした洗練されたデザイン（グラスモフィズム）を採用しながら、アセットの最適化により驚異的なロードスピードを実現。モバイルでもデスクトップでも、ストレスのないブラウジングが可能です。

---

## 🎯 こんなシーンにおすすめ

- **風景・天体カメラマン**: 最高の一枚を撮るための「月待ち」時間を正確に把握。
- **釣り愛好家**: 潮汐のリズムと月齢を組み合わせた、最適な釣行計画に。
- **ナイトアクティビティ**: 夜道を照らす月の光を計算し、安全で快適なアウトドアイベントを計画。
- **天体ファン**: 次に見られる天体ショーを、最も視認性の高い時間帯でチェック。
